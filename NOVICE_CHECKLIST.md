# Novice Setup Checklist

Tick these off in order.

## Firebase

- [ ] I opened Firebase project `lennys-training`.
- [ ] I enabled Authentication.
- [ ] I enabled Email/Password sign-in.
- [ ] I created Firestore Database, not Realtime Database.
- [ ] I chose Production mode.
- [ ] I pasted and published the Firestore rules.

## GitHub

- [ ] I created a public repo called `lenny-recall-app`.
- [ ] I uploaded the contents of the project folder.
- [ ] `index.html` is visible at the top level of the repo.
- [ ] I enabled GitHub Pages from `main` and `/root`.
- [ ] I copied the GitHub Pages URL.

## Firebase domain

- [ ] I added `YOUR-USERNAME.github.io` to Firebase Authentication > Settings > Authorised domains.

## Test

- [ ] I opened the GitHub Pages URL.
- [ ] I created an account in the Plan tab.
- [ ] I logged a few reps.
- [ ] I refreshed and the progress remained.
- [ ] I opened the app on another device and signed in.
- [ ] The progress appeared on the second device.

## iPhone

- [ ] I opened the app URL in Safari.
- [ ] I tapped Share > Add to Home Screen.
- [ ] I opened the app from the Home Screen icon.
- [ ] I signed in.
