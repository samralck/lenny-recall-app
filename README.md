# Lenny Recall App

A free, cross-device, iPhone-first training app for Lenny's 42-day recall rebuild.

## What this app does

- Guides the 42-day programme day by day.
- Logs each mini-session.
- Uses success/miss tracking for behaviour reps so the 80% rule is visible.
- Saves immediately to this device.
- Syncs across devices using Firebase Authentication and Firestore.
- Keeps Export/Restore backup as a safety net.
- Works as a static website on GitHub Pages.

## Free stack

- GitHub Pages for hosting.
- Firebase Spark plan for Authentication and Firestore.
- localStorage as local fallback.
- No paid domain, server, Node install, build system or credit card required.
